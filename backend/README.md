## Backend

* **backend:** это часть, отвечающая за отправку задач, полученных от фронта, через ansible и добавления пользователей в БД
* **зависимости:** backend использует данные, получаемые фронтенда и выполняет поставленные задачи отправляя их на компьютеры. Информацию от компьютеров получает автоматически и отправляет им публичный ключ ssh для связи через ansible

## Структура backend



 <pre>
    .
    ├── classroom_management                   # Корневая папка проекта
    │   ├── backend/                           # Все файлы для серверной части
    │   |   ├── ansible/                       # Директория ansible с данными о пользовтелей и самими задачами
    │   |   ├── apps.go                        # Предоставляет список доступных задачи фронту
    │   |   ├── install_server.sh              # Сборка сервеной части
    │   |   ├── main.go                        # Точка входа
    │   |   ├── register.go                    # Отвечает за регистрацию всех новых компьютеров
    │   |   ├── selected_programs.go           # Отвечает за принятие задач от фронта и отправки на компьютеры через ansible
    │   |   ├── sync_for_front.go              # Предоставляет список всех компьютеров фронту
    │   ├── ...                                # Другие папки
    │   ├── README.md                          # Описание проекта, в том числе этой части
</pre>

## Необходимые приложения перед запуском: golang
golang 1.25+

## Инструкция к использованию

### Установить репозиторий

<pre>git clone https://github.com/hharmen/classroom_management \
cd classroom_management/backend</pre>

Затем необходимо находясь в директории backend написать

<pre>
  chmod +x install_server.sh
  sudo ./install_server.sh
</pre>

Впринципе готово, после этого программа сама собирается и добавляется в сервис myserver, работает на порте 6000
