## agent

* **agent:** это часть, отвечающая за отправку данных о компьютере серверу и настройке ssh с добавлением ssh ключей для предоставления доступа серверу

## Структура agent



 <pre>
    .
    ├── classroom_management                   # Корневая папка проекта
    │   ├── agent/                             # Все файлы для агента
    │   |   ├── computer-agent.go              # Сам агент который постоянно отправляет свои данные серверу
    │   |   ├── install.sh                     # Сборка агента, скачивание необходимых пакетов и добавление в сервис для постоянной работы
    │   ├── ...                                # Другие папки
    │   ├── README.md                          # Описание проекта, в том числе этой части
</pre>

## Необходимые приложения перед запуском: golang
golang 1.25+

## Инструкция к использованию

### Установить репозиторий

<pre>git clone https://github.com/hharmen/classroom_management \
cd classroom_management/agent</pre>

### Сборка агента

Затем необходимо находясь в директории agent написать

<pre>
  chmod +x install.sh
  sudo ./install.sh
</pre>

### Ввод данных

Необходимо по очереди вести данные (название комнаты, название компьютера, айпи сервера, порт для подключения к серверу). В терминале будет видно что надо вести и когда

## Готово. Теперь вы можете настроить остальные части и все будет готово, агент уже посылает свои данные каждые 5 минут (если не получилось отправить, то каждые 3 секунды)
