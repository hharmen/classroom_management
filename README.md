## Front_End

* **front:** это часть, отвечающая за пользовательский интерфейс приложения
* **зачем:** необходима для отправки пользователем команд для установки необходимых приложений
* **зависимости:** front использует данные, получаемые от серверной части проекта. Для отдельного тестирования есть мини-сервер в `testing_server`. Его запуск аналогичен основному через `python mock_server.py` в его папке

## Структура front

 <pre>
    .
    ├── classroom_management                   # Корневая папка проекта

## Необходимые приложения перед запуском:

### Приложение запуска

Необходимо иметь компилятор/редактор для python кода. \
Например Visual Studio Code (https://code.visualstudio.com/download) или Pycharm (https://www.jetbrains.com/pycharm/)

### Необходимые библиотеки

*flask*, *json*, *os*, *requests*

установить библиотеку нужно в терминале при помощи команды `pip install <имя библиотеки>`

## Инструкция к использованию

### Установить репозиторий
```pip install -e git+https://github.com/hharmen/classroom_management``` \
```cd classroom_management```

Или для git bash: \
```git clone https://github.com/hharmen/classroom_management``` \
```cd classroom_management```

### Активировать виртуальную среду

Необходимо в соответствии с вашим компилятором создать виртуальную среду Python (рекомендуется среда .venv)

Далее активировать её при помощи команды `source .venv/Scripts/activate`

### Запустить
```cd front/front_files``` \
```python server.py``` 

### Перейти на страницу

После запуска перейти по ссылке в сообщении: 
`Основной сервер запущен на <ссылка>`
