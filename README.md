# Проект Система Удалённого Доступа СУД

## Функционал

Это комплекс программ для загрузки необходимых программ из веб формата на нужные компьютеры с установленными агентами

### Программа способна
* Загружать необходимые для работы программы на несколько компьютеров
* Отслеживать уже установленные программы и их версии

### Деление функционала

* **веб:** необходим для взаимодействия пользователя с программой. Позволяет выбирать подключённые компьютеры и готовые программы, и далее отправлять на сервер запрос на их загрузку
* **сервер:** хранит в себе программы для скачивания и связывает дочерные компьютеры с веб интерфейсом
* **агент:** отвечает за принятие файлов от сервера и загрузку на компьтер
